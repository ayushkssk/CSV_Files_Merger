CSV Merger Project Documentation

Overview

This project is a CSV Merger application that allows users to merge multiple CSV files into a single file. It is built using Flask and provides a simple web interface for users to upload and merge files. The application is designed to handle large datasets efficiently and provides a seamless experience for users needing to consolidate data from various sources.

How It Works

The CSV Merger web application operates through a straightforward process:

  1. Users access the web interface and are prompted to upload multiple CSV files.
  2. Once the files are uploaded, the application reads the contents of each CSV file.
  3. The application then merges the data from all uploaded files into a single CSV file. This involves combining rows from each file, ensuring that the headers are consistent.
  4. After merging, the application provides a download link for the user to obtain the consolidated CSV file.

Installation

  1. Clone the repository: git clone https://github.com/yourusername/CSV_Files_Merger.git
  2. Navigate to the project directory: cd CSV_Files_Merger
  3. Install the required packages: pip install -r requirements.txt

Usage

  1. Start the Flask server: python app.py
  2. Open your web browser and go to http://localhost:5001
  3. Upload the CSV files you want to merge and click the "Merge" button.

Features

  • Merge multiple CSV files into one.
  • Download the merged CSV file.
  • Simple and intuitive web interface.
